Thank you for downloading this section of Mr. Eat Gum Guy.
There are four different zip file each containing a separate
part of this freeware game. The different zip files are listed
below.

1. Main Game Zip
	- Contains Main Project, Forms, Modules, Sprites,
	  music files (midi), and some sound files.

2. Graphics and Sound Zip 1
	- Contains Graphics and sounds necessary for code.

3. Graphics and Sound Zip 2
	- Contains Graphics and sounds necessary for code.

4. Sound
	- Contains last of sound files needed for proper
	  running of the program.

Thanks for taking the time to get all of the files necessary
for this game.
